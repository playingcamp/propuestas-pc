TL;DR — Veredicto rápido

HubSpot Free/Starter + stack code-first (Apps Script + GitHub + APIs) gana a GHL para Playing Camp.

  • Coste arranque: $0–50/mes (HubSpot Free o Starter) vs $97/mes (GHL Starter)

2. Pros y contras detallados

VS

✓ HubSpot — Pros

  • Free tier real: 1.000 contactos, usuarios ilimitados, sin caducidad
  • API robusta: REST + webhooks bien documentada, fácil integrar Apps Script
  • Integración LinkedIn: Sales Hub conecta directo (no Zapier tax)
  • Inbox multi-bandeja: gestiona mayores@, empresas@, partnerships@ todas en Starter
  • Pipelines ilimitados en cualquier plan (incl. Free)
  • Custom Objects (Pro+) ideal para facilitadores futuro
  • Marketplace 1.500+ integraciones nativas
  • Stripe Billing nativo para facturas
  • Calidad email deliverability excelente
  • Comunidad/docs enormes, fácil aprendizaje

✗ HubSpot — Contras

  • Cliff brutal Starter→Pro: $50 → $890/mes (17x)
  • Onboarding obligatorio $3.000 en Pro, $7.000 Enterprise
  • Marketing contacts facturan por contacto activo (overages instantáneos)
  • Contratos anuales Pro/Enterprise (no escapas mid-year)
  • Landing pages capadas en Starter (solo formularios básicos)
  • Workflows avanzados = Pro ($890) — Starter solo "simple automation"
  • WhatsApp Chat Hub add-on $25/mes
  • AI credits limitados por plan, se consumen rápido
  • Reportes custom = Pro
  • Lead scoring = Pro

✓ GoHighLevel — Pros

  • Flat-fee predecible: $97 incluye casi todo
  • Contactos ilimitados en todos los planes
  • Usuarios ilimitados en todos los planes
  • WhatsApp nativo (sin add-on)
  • Landing pages + funnels drag-drop incluidos
  • SMS + llamadas nativos (Twilio integrado)
  • Calendario propio (no necesitas Calendly)
  • Reputation management (reviews automáticas)
  • Workflows visuales avanzados en plan base
  • White-label en SaaS Pro ($497) si quisieras revender

✗ GoHighLevel — Contras

  • Email deliverability mala: caídas de 30%+ en bandeja vs tools dedicados
  • Interfaz lenta/clunky, curva aprendizaje alta
  • Constantes bugs/breakdowns reportados en foros
  • API limitada vs HubSpot — integraciones custom más difíciles
  • LinkedIn integración débil: solo workaround vía Zapier
  • Soporte mediocre: tickets lentos, comunidad pequeña
  • Vendor lock-in: exportar data complicado
  • Costes uso impredecibles (SMS, AI, WhatsApp)
  • Marketing automation menos sofisticada
  • Mejor para agencias que para B2B propio (tu caso)

4. Workarounds para ahorrar dinero según escales

La gran ventaja para Playing Camp: ya tienes el stack code-first. Cada workaround sustituye una feature paga de HubSpot Pro.

Workaround #1: Landing pages → GitHub Pages + Cloudflare

HubSpot Pro feature: Landing pages con A/B testing ($890/mes)

Reemplazo: GitHub Pages (gratis) + HubSpot form embed

  • HTML/CSS custom, control 100% diseño
  • Videos YouTube/Vimeo embed sin límite
  • HubSpot form vía script tag
  • A/B testing: 2 branches en Git con redirect
  • SEO control total

Ahorro: ~$840/mes

Workaround #2: Workflows avanzados → Apps Script

HubSpot Pro feature: Workflows con custom code blocks ($890/mes)

Reemplazo: Apps Script + HubSpot API (gratis)

Tu setup actual ya tiene 6 scripts en producción. Patrón:

// Webhook receives event function doPost(e) { const data = JSON.parse(e.postData.contents); // Custom logic JSON-style const contact = HubSpot.findContact(data.email); if (contact.tag === 'partnerships') { HubSpot.moveDealStage(contact.dealId, 'qualified'); Gmail.sendTemplate('festival-followup', contact); } }

Ahorro: ~$840/mes

Workaround #3: Newsletter → MailerLite o Brevo

HubSpot Pro feature: Email marketing con segmentación avanzada

Reemplazo:

  • MailerLite: gratis hasta 1.000 contactos, $9/mes hasta 2.500
  • Brevo (ex-Sendinblue): gratis 300 emails/día, $9/mes hasta 20k/mes
  • Sync con HubSpot vía Zapier ($0-20/mes)

Ahorro: $830-880/mes

Workaround #4: Social media → Buffer/Metricool

HubSpot Pro feature: Social Hub publicación + monitoring

Reemplazo: Buffer ($15) o Metricool ($19)

  • Publica en LinkedIn, IG, FB, TikTok
  • Mejor analytics que HubSpot
  • Engagement → Apps Script → HubSpot tag

Ahorro: ~$870/mes

Workaround #5: Lead scoring → Apps Script + custom fields

HubSpot Pro feature: Predictive lead scoring

Reemplazo: Custom property en HubSpot + Apps Script calcula score nightly

// Daily cron via Apps Script function recalcLeadScores() { const contacts = HubSpot.getContacts({active: true}); contacts.forEach(c => { const score = (c.emailOpens * 5) + (c.formSubmissions * 20) + (c.linkedInVisits * 10); HubSpot.updateContact(c.id, {lead_score: score}); }); }

Ahorro: ~$840/mes

Workaround #6: AI agentes call summary → Fireflies + Apps Script

HubSpot Pro feature: AI call summarization (Sales Hub Pro)

Reemplazo: Fireflies.ai ($10) → webhook → Apps Script → HubSpot note

Ahorro: ~$80-200/mes

Workaround #7: Custom objects facilitadores → Companies object

HubSpot Pro feature: Custom Objects (Facilitadores schema dedicado)

Reemplazo: Usa Companies object con custom properties

  • Tag: "facilitador"
  • Custom props: especialidad, tarifa, zona, idiomas
  • Asocia Deals (talleres) → Companies (facilitadores)
  • Apps Script: matching automático

Ahorro: ~$840/mes

Workaround #8: Reportes custom → Looker Studio + HubSpot connector

HubSpot Pro feature: Custom reports dashboards (100 reportes)

Reemplazo: Looker Studio (gratis) + HubSpot API

  • Conector nativo HubSpot → Looker
  • Dashboards ilimitados
  • Comparte vía link

Ahorro: ~$840/mes

Workaround #9: n8n para orquestación visual

Cuándo usar n8n: Solo si necesitas workflows visuales no-code (equipo non-tech) o orquestación 10+ pasos

  • n8n self-hosted: $0 (VPS $5/mes)
  • n8n cloud: $20-50/mes
  • Webhook → n8n → HubSpot API + Apollo + Twilio + Calendly

Para tu caso: Apps Script gana. n8n añade complejidad sin valor extra.

6. Recomendación final Playing Camp

HubSpot Sales Hub Starter + Stack code-first

Razón: Combina lo mejor de ambos mundos:

  • HubSpot CRM/email/sync robusto y battle-tested
  • Tu Apps Script setup cubre el 80% de lo que pagarías en Pro
  • GitHub Pages reemplaza landing pages caras
  • Coste predecible (sin sustos contact-tier)
  • Escapas del cliff $50→$890/mes
  • LinkedIn integración nativa (vs GHL workaround)
  • Deliverability email superior a GHL

Plan acción próximos 30 días:

  1. Crear cuenta HubSpot Free CRM con empresas@playingcamp.es
  2. Conectar Gmail (mayores@, empresas@, partnerships@)
  3. Configurar 3 pipelines (1 por vertical)
  4. Migrar Apps Scripts existentes → webhook HubSpot
  5. Crear primera landing GitHub Pages + form HubSpot embedded
  6. Decidir entre Free (1.000 contactos) o Starter ($15) según volumen actual

Cuándo SÍ considerar GHL

  • Si pivotas a modelo agencia revendiendo a facilitadores como sub-cuentas (SaaS Pro)
  • Si SMS/WhatsApp marketing se vuelve core (no email)
  • Si no quieres tocar código (pero tú sí quieres → no aplica)

Cuándo upgrade a HubSpot Marketing Pro

Solo cuando todos estos puntos se cumplan:

  • Tienes 3.000+ contactos activos en marketing
  • Necesitas A/B testing nativo (no GitHub branches)
  • Equipo 3+ personas que necesitan UI no-code
  • Facturas mensuales >$5.000 que justifican $890/mes
  • Tiempo de tu equipo > tiempo programación workarounds
